SHA512 hash procedures.<br /><br />These stack-returning core wrappers call `::miden::precompiles::crypto::hashes::sha512` for event<br />handling and deferred verification.<br />


## miden::core::crypto::hashes::sha512
| Procedure | Description |
| ----------- | ------------- |
| hash_bytes | Computes SHA512 hash of data stored in memory.<br /><br />Input:  [ptr, len_bytes, ...]<br />Output: [DIGEST_U32[16], ...]<br /> |
| hash | Computes SHA512 hash of a single 256-bit input.<br /><br />Input:  [INPUT_U32[8], ...]<br />Output: [DIGEST_U32[16], ...]<br /> |
| merge | Merges two 256-bit inputs via SHA512 over their 512-bit concatenation.<br /><br />Input:  [INPUT_L_U32[8], INPUT_R_U32[8], ...]<br />Output: [DIGEST_U32[16], ...]<br /> |
