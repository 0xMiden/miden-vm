Expand description

A ring implements InjectiveMonomial<N> if the algebraic function f(x) = x^N is an injective map on elements of the ring.

We do not enforce that this map be invertible as there are useful cases such as polynomials or symbolic expressions where no inverse exists.

However, if the ring is a field with order q or an array of such field elements, then f(x) = x^N will be injective if and only if it is invertible and so in such cases this monomial acts as a permutation. Moreover, this will occur exactly when N and q - 1 are relatively prime i.e. gcd(N, q - 1) = 1.
