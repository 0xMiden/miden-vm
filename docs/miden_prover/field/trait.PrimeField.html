Expand description

A field isomorphic to ℤ/p for some prime p.

There is a natural map from ℤ to ℤ/p which sends an integer r to its conjugacy class [r]. Canonically, each conjugacy class [r] can be represented by the unique integer s in [0, p - 1) satisfying s = r mod p. This however is often not the most convenient computational representation and so internal representations of field elements might differ from this and may change over time.
