Expand description

Represents the set of primitive instructions in Miden Assembly syntax.

NOTE: For control flow instructions, see crate::ast::Op.

pub const fn has_textual_representation(&self) -> bool

Returns true if this instruction has a textual representation in Miden Assembly.

Some instructions (like DebugVar) are compiler-internal and have no surface syntax. They should be skipped during pretty-printing.
