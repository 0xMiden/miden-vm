Synchronous wrapper for the async prove() function.

This method is only available on non-wasm32 targets. On wasm32, use the async prove() method directly since wasm32 runs in the browser’s event loop.

§Panics

Panics if called from within an existing Tokio runtime. Use the async prove() method instead in async contexts.
